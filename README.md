# BarakahFlow Landing Page

Premium, minimalist landing page for the BarakahFlow Android application. Designed with a focus on Islamic finance, Zakat calculation, and modern mobile-first aesthetics.

## Features

- **Premium Design**: 100% centered typography with a "glassmorphic" aesthetic.
- **Smart App Previews**: High-resolution floating screenshots from the Samsung Galaxy S25.
- **PWA Support**: Built-in "Smart Install Prompt" for iOS users without an App Store release.
- **PWA Manifest**: Standardized `manifest.json` for home screen integration.
- **Halal Tools**: Showcasing Smart Budgeting, Zakat calculators, and gold-tracking features.

## Tech Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Variables).
- **Icons**: Lucide Icons (via CDN).
- **Fonts**: Montserrat (Google Fonts).

## Deployment

Optimized for 1-click deployment to **Vercel** or **GitHub Pages**.
